# GodoFredo.Ninja

## Structure

```
├───app
│   ├───controllers
│   ├───routes
│   └───views
│       ├───includes
│       ├───layouts
│       └───pages
├───config
│   └───env
└───public
    ├───css
    ├───fonts
    ├───img
    ├───lib
    ├───scripts
    │   ├───config
    │   ├───controllers
    │   ├───directives
    │   ├───filters
    │   ├───services
    │   └───views
    └───stylus
```

## Usage

### Install global
```
$ npm install -g stylus
$ npm install -g gulp
$ npm install -g bower
```

### Install local
```
$ npm install
$ bower install
```
### Development mode
```
$ node server
$ gulp
```
